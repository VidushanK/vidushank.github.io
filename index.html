How I started?

My Father introduced me to Geocities and Angelfire which sparked my interest in Web development, soon after graduating highschool I was pursing a degree in Computer Science

Soon after enrolling in College, I came to the realization that learning the theory about computers and programming isn't what I wanted to learn at the time. Which led me into taking a break from College and enrolling in Lighthouse Labs which led me into becoming a Freelance Web Developer!

Why?

I develop fast interactive full-stack web applications, using the latest technologies and techniques to reduce page load time and improve user experience. I use my knowledge and tools to save time in development.
